Estrutura Básica do Grid

Para criar um layout Grid, primeiro devemos definir um contêiner que irá funcionar como uma grade. Dentro deste contêiner, os itens se alinham conforme as colunas e linhas definidas. 

            
    div class="grid-container" 
        div class="item item1" Item 1 /div
        div class="item item2" Item 2 /div
        div class="item item3 borda-redonda" Item 3 /div
        div class="item item4" Item 4 /div
    div>

    .grid-container {
        display: grid; /* Define o container como grid */
        grid-template-columns: 200px 200px; /* Define duas colunas de 200px cada */
        grid-template-rows: 150px 150px;
        gap: 10px;
        background-color: lightgray;
    }

Ajustando Colunas e Linhas com o fr

Unidade fr (fraction unit) é muito útil se deseja criar colunas ou linhas proporcionais, distribuindo o espaço disponível entre os itens.
